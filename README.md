# Discord Chan (Go)

## Summary:

Discord Chan is a discord bot written in Go. 
It is being used as a fun way to learn Golang.

## Features:

#### Implemented
- Weather (using Weatherstack API)
- Roll
- Echo
- FOAAS (for ping purposes)
- Woof (Random Dog Pictures)
- Gif (Gifs using GIPHY API)

Deployed via Heroku
